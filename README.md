# Palindrome_detector_py
<br>
[ENG] Program that reads a sentence and verify if it's a Palindrome, Ex:
<br>
- Ana
<br>
- Taco cat
<br>
- Was it a cat I saw
<br>
- Mom
<br>
- No lemon no melon
<br>
